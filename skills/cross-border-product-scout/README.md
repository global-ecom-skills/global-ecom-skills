# Cross-Border Product Scout

跨境电商选品智能体 - 发现 Amazon→Temu→1688 套利机会

## 功能特点

- 🔍 **Amazon 热销榜分析** - 自动获取各类目热销商品数据
- 📊 **Temu 竞品对比** - 分析 Temu 上同类商品的价格和销量
- 💰 **1688 供应商比价** - 查找国内供应商，计算采购成本
- 📈 **利润空间计算** - 自动计算毛利率和预期利润
- 📝 **选品报告生成** - 生成包含完整数据的分析报告

## 安装

```bash
npx skills install global-ecom-skills/global-ecom-skills/cross-border-product-scout
```

## 使用方法

### 基础用法

```
用户: 帮我分析厨房用品类目的选品机会
Agent: [执行选品分析，生成报告]
```

### 指定商品分析

```
用户: 分析这个商品的套利空间：[Amazon 商品链接]
Agent: [分析该商品在各平台的价格差异]
```

### 批量分析

```
用户: 分析 Amazon 厨房类目 Top 10 热销品的套利机会
Agent: [批量分析并生成对比报告]
```

## 前置条件

- 需要浏览器支持（用于访问电商平台）
- 建议先使用 `platform-auto-login` skill 登录各平台

## 限制

- 1688 搜索可能需要手动完成验证码
- 价格数据可能有延迟
- 部分平台可能有访问限制
