# Temu Listing Optimizer

Optimize your Temu product listings for maximum visibility and sales.

## Description

This skill helps sellers create high-converting Temu listings by:
- Crafting compelling titles with keywords
- Writing persuasive product descriptions
- Optimizing images and bullet points
- Pricing strategy recommendations

## Usage

Activate when user wants to:
- Create new Temu listings
- Optimize existing listings
- Improve conversion rates
- Research Temu keywords

## Workflow

1. **Product Analysis** - Understand product features and benefits
2. **Keyword Research** - Find high-traffic, low-competition keywords
3. **Title Optimization** - Create keyword-rich, compelling titles
4. **Description Writing** - Write benefit-focused descriptions
5. **Price Positioning** - Recommend competitive pricing

## Best Practices

- Title: 80-120 characters, front-load keywords
- Images: White background, lifestyle shots, size reference
- Description: Benefits > Features, use bullet points
- Price: Research competitors, consider promotions

## Output Format

```markdown
## Optimized Listing: [Product Name]

### Title (Optimized)
[Keyword-rich title here]

### Key Features (5 bullets)
1. [Benefit-focused bullet]
2. [Benefit-focused bullet]
...

### Description
[Compelling product description]

### Pricing Recommendation
- Suggested Price: $XX.XX
- Competitor Range: $XX - $XX
```
