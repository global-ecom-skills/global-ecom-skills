# 品类选择指南

## 推荐品类详解

### 🟢 低门槛高潜力品类

#### 1. 家居收纳
- **热门子类**：衣柜收纳盒、桌面整理、厨房收纳架
- **亚马逊价格带**：$10-$30
- **1688 拿货价**：¥5-¥30
- **资质要求**：无
- **注意点**：注意尺寸和重量，影响运费

#### 2. 厨房小工具
- **热门子类**：切菜器、开瓶器、量勺、硅胶模具
- **亚马逊价格带**：$8-$25
- **1688 拿货价**：¥3-¥20
- **资质要求**：食品接触材料需 FDA（硅胶、不锈钢通常已有）
- **注意点**：避免电动产品

#### 3. 宠物用品
- **热门子类**：宠物玩具、梳子、牵引绳、宠物服装
- **亚马逊价格带**：$10-$40
- **1688 拿货价**：¥5-¥30
- **资质要求**：无（避开宠物食品）
- **注意点**：美国宠物市场巨大，复购率高

#### 4. 汽车配件
- **热门子类**：手机支架、遮阳板、座椅收纳、清洁工具
- **亚马逊价格带**：$10-$35
- **1688 拿货价**：¥5-¥25
- **资质要求**：无（避开安全相关配件）
- **注意点**：注意车型兼容性描述

#### 5. 办公文具
- **热门子类**：桌面收纳、便签本、文件夹、笔筒
- **亚马逊价格带**：$8-$25
- **1688 拿货价**：¥3-¥15
- **资质要求**：无
- **注意点**：开学季需求旺盛

#### 6. 户外露营
- **热门子类**：野餐垫、便携餐具、收纳袋、小型工具
- **亚马逊价格带**：$15-$50
- **1688 拿货价**：¥10-¥40
- **资质要求**：无
- **注意点**：季节性强，春夏为旺季

#### 7. 手机配件
- **热门子类**：手机壳、支架、数据线收纳、屏幕清洁
- **亚马逊价格带**：$8-$20
- **1688 拿货价**：¥2-¥10
- **资质要求**：无（避开充电器、电池）
- **注意点**：更新快，注意手机型号

#### 8. 服装配饰
- **热门子类**：帽子、围巾、发饰、包包
- **亚马逊价格带**：$10-$30
- **1688 拿货价**：¥5-¥25
- **资质要求**：无
- **注意点**：注意尺码和材质描述

## 🔴 高门槛品类（避开）

### 食品/保健品
- **所需资质**：FDA 注册、食品经营许可证、营养标签
- **风险**：海关扣押、法律责任
- **建议**：完全避开

### 医疗器械
- **所需资质**：FDA 510(k)、医疗器械注册证
- **风险**：高额罚款、产品召回
- **建议**：完全避开

### 儿童用品
- **所需资质**：CPC 认证、CPSIA 合规、ASTM 测试
- **风险**：测试费用高（$500-$2000/款）
- **建议**：除非有稳定供应商已有认证

### 电子产品
- **所需资质**：FCC、CE、UL 认证
- **风险**：认证成本高、售后问题多
- **建议**：避开，除非是简单配件

### 化妆品
- **所需资质**：FDA 注册、成分备案、标签合规
- **风险**：成分问题、过敏投诉
- **建议**：完全避开

### 玩具
- **所需资质**：ASTM F963、EN71、CPC
- **风险**：测试费用高、召回风险
- **建议**：避开，除非供应商已有认证

### 电池类产品
- **所需资质**：UN38.3、MSDS、航空运输限制
- **风险**：物流限制、安全问题
- **建议**：完全避开

## 品类选择决策树

```
商品是否涉及人体接触/摄入？
├── 是 → 🔴 高风险，建议避开
└── 否 → 继续评估
    │
    商品是否需要电源/电池？
    ├── 是 → 🟡 中风险，需要认证
    └── 否 → 继续评估
        │
        商品是否面向儿童？
        ├── 是 → 🔴 高风险，需要 CPC
        └── 否 → 继续评估
            │
            商品是否涉及安全功能？
            ├── 是 → 🟡 中风险，需要评估
            └── 否 → 🟢 低风险，可以考虑
```
