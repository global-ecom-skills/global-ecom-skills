# Cross-Border Arbitrage

Find profitable arbitrage opportunities between platforms.

## Description

This skill identifies price gaps between e-commerce platforms for profitable arbitrage:
- Amazon to Temu opportunities
- 1688 to international platforms
- Regional price differences
- Seasonal arbitrage windows

## Usage

Activate when user wants to:
- Find arbitrage opportunities
- Compare cross-platform prices
- Calculate profit margins
- Identify trending products for resale

## Workflow

1. **Source Platform Analysis** - Find low-cost products (1688, Amazon)
2. **Target Platform Research** - Check prices on Temu, eBay, etc.
3. **Margin Calculation** - Factor in all costs
4. **Risk Assessment** - Evaluate competition and demand
5. **Opportunity Ranking** - Score and prioritize opportunities

## Profit Calculation

```
Profit = Selling Price - (Source Cost + Shipping + Platform Fees + Customs)
ROI = (Profit / Total Investment) × 100%
```
