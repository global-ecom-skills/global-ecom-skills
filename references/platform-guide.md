# 平台操作指南

## 亚马逊数据获取

### Best Sellers 页面结构

**URL 格式**：
- 总榜：`amazon.com/Best-Sellers/zgbs`
- 分类榜：`amazon.com/Best-Sellers-[Category]/zgbs/[category-id]`

**关键数据点**：
1. 商品标题
2. 价格
3. BSR 排名（Best Seller Rank）
4. 评论数量
5. 评分星级
6. 商品 ASIN

### Movers & Shakers（飙升榜）

**URL**：`amazon.com/gp/movers-and-shakers`

**特点**：
- 显示过去 24 小时销量增长最快的商品
- 包含增长百分比
- 适合发现新趋势

### 数据解读

| BSR 范围 | 日销量估算 | 市场热度 |
|----------|-----------|----------|
| 1-100 | 1000+ | 🔥 爆款 |
| 100-1000 | 100-1000 | 🔥 热卖 |
| 1000-10000 | 10-100 | ✅ 稳定 |
| 10000-50000 | 1-10 | ⚠️ 一般 |
| 50000+ | <1 | ❌ 冷门 |

## Temu 市场分析

### 搜索技巧

1. **关键词翻译**：将中文商品名翻译成英文搜索
2. **多关键词测试**：同一商品可能有多种叫法
3. **查看相关搜索**：发现用户真实搜索习惯

### 竞争度判断

**搜索结果数量**：
- < 50 个：🟢 蓝海，机会大
- 50-200 个：🟡 有竞争，需差异化
- 200-500 个：🟠 竞争激烈
- > 500 个：🔴 红海，谨慎进入

**头部卖家销量**：
- 头部销量 < 500：市场未成熟
- 头部销量 500-2000：市场成长期
- 头部销量 > 2000：市场成熟期

### 价格策略

Temu 用户对价格敏感，定价建议：
- 比亚马逊低 40-60%
- 比 Temu 竞品低 5-10%（初期引流）
- 保持 30%+ 毛利空间

## 1688 供应商筛选

### 搜索入口

- **主站**：`1688.com`
- **跨境专供**：`waimaoquan.alibaba.com`（有出口经验的供应商）

### 供应商筛选标准

**优先选择**：
- ✅ 实力商家（平台认证）
- ✅ 源头工厂（非贸易商）
- ✅ 3 年以上店龄
- ✅ 回头率 > 30%
- ✅ 有跨境经验

**注意事项**：
- ⚠️ 起订量（MOQ）是否可接受
- ⚠️ 是否支持定制/贴牌
- ⚠️ 发货时效
- ⚠️ 是否有现货

### 价格谈判技巧

1. **阶梯报价**：询问不同数量的价格
2. **样品先行**：先拿样品验证质量
3. **长期合作**：表达长期合作意向争取更好价格
4. **多家比价**：至少联系 3 家供应商

### 成本计算

```
总成本 = 商品成本 + 国内运费 + 国际运费 + 平台费用

示例：
- 商品成本：¥20
- 国内运费：¥3
- 国际运费：¥15（约 500g）
- 平台费用：15% × 售价

如果 Temu 售价 $10（约 ¥72）：
- 平台费用：¥10.8
- 总成本：¥48.8
- 毛利：¥23.2（32%）
```

## 利润计算公式

### 基础公式

```
毛利 = Temu 售价 - 商品成本 - 运费 - 平台费用
毛利率 = 毛利 / Temu 售价 × 100%
```

### 目标毛利率

- 🟢 > 40%：优秀，值得做
- 🟡 30-40%：可以，有空间
- 🟠 20-30%：勉强，需要走量
- 🔴 < 20%：不建议，风险高

### 定价策略

```
建议 Temu 售价 = 亚马逊售价 × 0.5
目标拿货价 = Temu 售价 × 0.3
```

这样可以保证：
- 比亚马逊便宜 50%（吸引用户）
- 保持 30%+ 毛利空间
