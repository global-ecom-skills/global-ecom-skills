# UI Optimizer

产品页面 UI 优化智能体 - 基于 Google Material Design 规范

## 功能特点

- 🎨 **Material Design 规范** - 严格遵循 Google 设计规范
- 🎯 **对标 skills.sh** - 以顶级产品为参考基准
- 📱 **响应式适配** - 完美支持移动端和 PC 端
- ⚡ **可落地方案** - 输出具体参数，可直接用于开发

## 安装

```bash
npx skills install global-ecom-skills/global-ecom-skills/ui-optimizer
```

## 使用方法

### 基础用法

```
用户: 请优化这个页面的 UI: [页面路径或 URL]
Agent: [分析页面，输出优化方案并实施]
```

### 指定优化范围

```
用户: 只优化导航栏和卡片组件的样式
Agent: [针对性优化指定模块]
```

## 设计规范摘要

### 色彩
- 主色调: ≤2 种
- 中性色: #FFFFFF, #F5F5F5, #EEEEEE, #333333, #666666, #999999
- 主色调参考: #1976D2 (蓝) 或 #00D4AA (青绿)

### 字体
- 字体族: Roboto, Inter, system fonts
- 标题: 18-24px, 600
- 正文: 14-16px, 400
- 辅助: 12-13px, 400

### 间距
- 模块间距: ≥24px
- 内边距: ≥16px
- 卡片间距: ≥16px

### 阴影
- 浓度: ≤10%
- 模糊半径: ≤4px

## 核心原则

1. **简洁** - 去除冗余装饰
2. **清晰** - 信息层级分明
3. **克制** - 色彩和元素克制
4. **易用** - 交互反馈清晰

## 禁忌

- ❌ 复杂渐变
- ❌ 厚重阴影
- ❌ 高饱和撞色
- ❌ 信息拥挤
- ❌ 复杂动画 (>300ms)
