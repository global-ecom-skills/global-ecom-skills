# 平台自动登录 Skill 🔐

帮助用户登录跨境电商平台（Temu、Amazon、1688），保持登录状态以便数据采集。

## 支持平台

| 平台 | 登录方式 | 所需信息 |
|------|----------|----------|
| Temu | 邮箱/手机号 | 邮箱或手机号 + 密码 |
| Amazon | 邮箱 | 邮箱 + 密码 |
| 1688 | 扫码/手机号 | 手机号 + 密码（或扫码） |

## 使用方法

直接对 AI 说：

```
帮我登录 Temu
帮我登录 Amazon
帮我登录 1688
帮我登录所有跨境平台
检查各平台登录状态
```

## 登录信息

登录时 AI 会询问你：
1. 登录方式（邮箱/手机号/扫码）
2. 账号（邮箱或手机号）
3. 密码（临时输入，不会保存）

## 安全说明

- ✅ 密码仅临时使用，不会保存
- ✅ 使用独立浏览器配置，与日常浏览器隔离
- ✅ 登录成功后保持 Cookie，下次无需重新登录
- ⚠️ 遇到验证码/滑块需要你手动协助完成
