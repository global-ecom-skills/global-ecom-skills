跨境电商 AI Agent Skills

帮助你发现热销、高利润的跨境电商选品机会。一键安装，增强你的 AI Agent 能力。

$ npx skills install global-ecom-skills/global-ecom-skills

📦 可用 Skills

cross-border-product-scout 热门

跨境电商选品智能体。发现亚马逊热卖品并搬运到 Temu 销售的套利机会。 支持 Amazon 热销榜分析、Temu 竞品对比、1688 供应商比价。

🏷️ 选品分析 📊 利润计算 🔍 市场调研
platform-auto-login 工具

跨境电商平台自动登录。支持 Temu、Amazon、1688 等平台的自动登录， 保持登录状态以便后续数据采集和操作。

🔐 自动登录 🌐 多平台 🤖 浏览器控制
