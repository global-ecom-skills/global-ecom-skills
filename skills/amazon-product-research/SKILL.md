# Amazon Product Research

Find winning products on Amazon with data-driven analysis.

## Description

This skill helps you discover high-potential products on Amazon by analyzing:
- Best Seller Rank (BSR) trends
- Review velocity and sentiment
- Price history and competition
- Seasonal demand patterns

## Usage

Activate when user wants to:
- Find profitable Amazon products
- Analyze product competition
- Research niche opportunities
- Validate product ideas

## Workflow

1. **Identify Category** - Select target category or niche
2. **Gather Data** - Collect BSR, reviews, pricing data
3. **Analyze Competition** - Evaluate existing sellers
4. **Score Opportunity** - Rate product potential (1-10)
5. **Generate Report** - Provide actionable insights

## Output Format

```markdown
## Product Analysis: [Product Name]

**Opportunity Score:** X/10

### Key Metrics
- BSR: #XXX in [Category]
- Reviews: XXX (X.X avg rating)
- Price Range: $XX - $XX
- Monthly Sales Est: XXX units

### Competition Analysis
- Number of sellers: XX
- FBA vs FBM ratio: XX%
- Brand dominance: Low/Medium/High

### Recommendation
[Buy/Pass/Research More]
```
