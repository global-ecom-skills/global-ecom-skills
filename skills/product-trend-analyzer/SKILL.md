# Product Trend Analyzer

Analyze product trends and predict market demand.

## Description

This skill helps identify trending products and predict future demand by:
- Analyzing search trends (Google Trends, platform data)
- Monitoring social media buzz
- Tracking seasonal patterns
- Identifying emerging niches

## Usage

Activate when user wants to:
- Find trending products
- Predict seasonal demand
- Identify emerging niches
- Validate product timing

## Data Sources

- Google Trends
- Amazon Movers & Shakers
- TikTok/Instagram trending products
- Platform search suggestions
- News and social media

## Workflow

1. **Trend Detection** - Identify rising search/sales patterns
2. **Validation** - Cross-reference multiple data sources
3. **Timing Analysis** - Determine trend lifecycle stage
4. **Competition Check** - Assess market saturation
5. **Recommendation** - Provide actionable insights

## Trend Lifecycle

```
Early → Growth → Peak → Decline → Stable/Dead
  ↑        ↑       ↑
 Best    Good    Risky
```

## Output Format

```markdown
## Trend Report: [Product/Niche]

### Trend Status
- Stage: Early/Growth/Peak/Decline
- Momentum: Rising/Stable/Falling
- Confidence: XX%

### Data Points
- Google Trends: +XX% (30 days)
- Social Mentions: XXX/week
- Platform BSR Change: -XX%

### Timing Recommendation
[Enter Now / Wait / Too Late]

### Related Trends
- [Related product 1]
- [Related product 2]
```
